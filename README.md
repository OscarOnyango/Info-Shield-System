Multi-Module Service Platform

## About
A multi-module system integrating:
 **Transport booking**
 **ICT equipment ordering**
 **Affordable housing reservations**

The platform includes **role-based access control** and management features for admins, staff, and users.

## Features
- User registration and login
- Role-based dashboards (Admin, Staff, User)
- Transport booking system (view, book, cancel)
- ICT equipment ordering (request, approve, track)
- Affordable housing reservations
- Data management and reporting for admins

## Technologies Used
- Frontend: HTML, CSS, JavaScript 
- Backend: Python
- Database: PostgreSQL
- Hosting / Deployment: Localhost
